## 🛰 시장 레이더 리포트 — Memsum

조사일: 2026-05-28 (이름 확정 2026-05-31)

> **이름**: Memsum (멤섬) — Memory + Summary
> **전략**: 카톡 통합 X, iOS 네이티브 앱 + 스크린샷 자동 감지 + 캘린더 OCR 시그니처
> **자세히**: [[아이디어/Memsum/브랜드|브랜드 가이드]]

### 판정

**🟡 틈새 시장 (iOS 네이티브 + 한국 시장 특화 조건)**
**이유**: 글로벌 mymind·Mem 2.0이 iOS 앱 + 일부 자동 캡처를 제공하지만, **"스크린샷 → 한국어 OCR → 구글 캘린더 자동 등록"** 결합 + 한국 시장 가격(₩4,900~)으로 진입 가능. 시그니처 기능(캘린더 OCR)이 핵심 차별점.

---

### 글로벌 경쟁사

#### A. iOS 네이티브 + 자동 캡처 (가장 가까운 경쟁자)

| 서비스명 | 국가 | 수익/투자 | 가격 | 캘린더 연동 | iOS 자동 감지 | Memsum과의 차이 |
|----------|------|-----------|------|-------------|--------------|------------------|
| **mymind** | 미국 | 소규모 | $13/월(~₩17,000) | ❌ | 부분 (공유 시트) | **가장 비슷**. 단 한국어 OCR 약함, 캘린더 연동 X, 가격 3배 |
| **Mem 2.0** | 미국 | $40M 투자, 리빌드 | $14.99/월 | ❌ | 부분 (음성 캡처) | AI 정리 자동화. 단 한국어 약함, 캘린더 X |
| **Apple Notes** | 미국 (Apple) | 무료 | 무료 | ⚠️ Apple Calendar 부분 | ✅ 시스템 통합 | 무료지만 큐레이션·요약 약함. 구글 캘린더 안 됨 |
| **Reflect Notes** | 미국 | 미공개 | $10/월 | ❌ | ❌ (수동) | 데일리 노트 중심, 자동 캡처 약함 |
| **Sumind.app** | 미국 | 미공개 | 미공개 | ❌ | 미상 | **컨셉 90% 동일** — "Stop saving links you'll never read. We summarize." → **강력한 시장 검증 증거**. 단 한국어·캘린더 약함 |

**Memsum의 명확한 시그니처**: **스크린샷 안의 한국어 날짜 → 구글 캘린더 자동 등록 제안** — 이 결합은 **현재 시장에 없음**

#### B. Read-Later + 하이라이트 (인접 시장)

| 서비스명 | 가격 | 차별점 |
|----------|------|--------|
| Readwise + Reader | $9.99/월 | 긴 글 + Resurfacer 강자. 단 한국 콘텐츠 약함, 사진/이미지 캡처 약함 |
| Glasp | $10/월 | 하이라이트 + 데일리 리서피싱 |
| Matter | $8/월 | 읽기 경험 우수, AI 약함 |

**갭**: 모두 "긴 글 읽기" 중심. 짧은 시각적 캡처(스크린샷·사진)에는 약함.

#### C. 노트·워크스페이스 (Anti-persona의 영역)

| 서비스명 | 차별점 vs Memsum |
|----------|-------------------|
| Notion | 거대 워크스페이스, 사용자 정리 100%. Anti-persona(정노션) 영역 |
| Obsidian | 마크다운 자유도, 사용자 정리 100%. Anti-persona 영역 |
| Heptabase | 시각적 캔버스, 리서치 중심 |

### 한국 시장 현황

- **iOS 네이티브 PKM 한국어 특화**: **부재**
- **스크린샷 자동 감지 한국 앱**: 사진 정리 앱 일부 있으나 OCR·캘린더·큐레이션 결합 부재
- **한국식 캘린더 OCR**: 글로벌 빅테크 약점, 한국 스타트업 부재 → **명백한 공백**

| 서비스/카테고리 | 평가 |
|-----------------|------|
| 노션 (한국 사용자 80만+) | 정리 사용자 100% 부담. Anti-persona |
| 옵시디언 (한국 사용자 5만+) | 마크다운 헤비유저용. Anti-persona |
| Apple 사진 앱 + Live Text | iOS 자체 OCR 있으나 정리·큐레이션·캘린더 자동 등록 없음 |
| 네이버 메모 / 카카오톡 메모 | 단순 메모, 검색·요약·자동화 없음 |
| **Slid** | 영상 노트 특화 (B2B 피벗) |
| **한국형 mymind/Mem** | 부재 |
| **스크린샷 → 한국 캘린더 자동 등록 앱** | **부재** ← Memsum 진입점 |

**핵심 발견**:
1. 한국 iPhone 사용자 1,500만+ (점유율 30%), 그 중 헤비 콘텐츠 소비자 약 300만
2. 한국 사용자는 카톡·인스타·유튜브·웹 스크린샷을 매일 5~20장 찍음 — 사진 앨범에 누적 1,000장+
3. 그 스크린샷 안의 **약속·일정·정보**가 다 휘발됨 → 이걸 OCR + 캘린더로 살리는 제품 부재
4. **이게 진짜 진입 포인트**

### 시장 규모 추정

- **글로벌 TAM**: AI 노트·캡처 도구 시장 약 $8B (2026, 22% CAGR)
- **글로벌 SAM**: iOS PKM/캡처 유료 사용자 2,000만~3,000만
- **한국 잠재 고객**:
  - 한국 iPhone 헤비 콘텐츠 소비자: 300만 명
  - "스크린샷 앨범 1,000장+" 사용자: 200만+
  - 디지털 마케터·1인 사업가·콘텐츠 크리에이터: 50만+
- **참고 근거**: 한국 스마트폰 통계청, Apple Korea 사용자 발표, 콘텐츠진흥원 크리에이터 통계

### ⚠️ 대기업 위험 분석 (재평가)

| 대기업 | 위험 시그널 | 대응 |
|--------|-------------|------|
| **Apple** (Apple Intelligence, Live Text 강화) | iOS 자체 OCR·요약 강화 진행 중. 가장 큰 위협 | 한국어 OCR + 한국 캘린더 양식 + 큐레이션 결합으로 차별화 |
| **Google** (Pixel Magic, Google Lens) | Android 강화 중 (Memsum iOS 우선이라 단기 영향 적음) | iOS 시장 우선 + Android 후속 |
| **mymind 한국 진출** | $13 → ₩9,900 한국 가격 인하 가능 | 가격 ₩4,900 + 한국어 정확도로 방어 |
| **OpenAI ChatGPT Memory** | 메모리 기능 강화 | 사용자 콘텐츠 큐레이션 X, 직접 충돌 아님 |
| **Sumind.app 한국 진출** | 동일 컨셉 글로벌 앱이 한국 시장 진입 | 한국어 + 캘린더 OCR + iOS 친화로 방어 |

**판단**: Apple Intelligence가 가장 큰 위협. 다만 Apple은 "구글 캘린더" 같은 경쟁사 서비스 통합에 미온적이므로, **"스크린샷 → Google Calendar 자동 등록"** 시그니처는 Apple이 직접 안 함 → 모트 가능.

### 🚨 한 번에 풀 빌드 가는 결정의 리스크 분석

이 전략(V1/V2 분할 없이 한 번에 네이티브 앱)의 명확한 리스크와 완화책:

#### 리스크 1 — PMF 검증 전 큰 투자
- **시그널**: 12주 빌드 후 출시했는데 다운로드 < 1,000 / Lite 결제 < 50
- **완화**: **빌드 시작 전 4주간 사전 판매 검증** — 데모 영상 + Figma 프로토타입 → 인스타 릴스·디스콰이엇 → 사전 결제 100명 미달 시 빌드 중단

#### 리스크 2 — Apple Intelligence 빠른 추격
- **시그널**: WWDC 2026/2027에서 Apple이 스크린샷 OCR + 캘린더 자동 등록 발표
- **완화**: 한국어 OCR 모트 + 구글 캘린더 통합 (Apple 안 함) + Pro 플랜(Ask Your Brain)으로 가치 확장

#### 리스크 3 — 앱 다운로드 마찰
- **시그널**: 디스콰이엇·인스타에서 본 사람의 다운로드 전환율 < 10%
- **완화**: 강한 인스타 릴스 후킹 + 첫 화면 가치 즉시 보여주기 + 첫 7일 무료

#### 리스크 4 — Mem AI 길 따라감
- **시그널**: 기능 야심병 → PMF 못 찾고 ARR 정체
- **완화**: 출시 시 기능 5개로 제한 (스크린샷 자동 + 캘린더 OCR + 공유 시트 + 자동 분류 + 주간 5줄), 이후 사용자 데이터로 확장

#### 리스크 5 — Sumind.app 직접 경쟁
- **시그널**: 동일 컨셉 영문 앱이 한국 시장 진출 또는 마케팅 강화
- **완화**: 한국어 OCR + 캘린더 OCR + ₩4,900 가격 + iOS 친화 + 멤섬 한국 친화 브랜드로 차별화

### 진입 추천 여부

**조건부 추천** ✅ (조건 충족 시 강력 추천)

#### 추천 이유
1. **수요 검증 완료** — mymind·Mem·Readwise·Sumind 합산 검증
2. **한국어 캘린더 OCR 결합 = 시장에 없음** (시그니처 차별점)
3. **한국 iPhone 헤비유저 300만 = 충분한 대중 시장**
4. **용량 기반 가격 = 자산 누적 락인** + 마진 안정

#### 필수 조건 (Pre-launch Must)
1. **사전 판매 검증 4주 우선** — 빌드 시작 전 사전 결제 100명 모집 (인스타 릴스 + 데모 영상)
2. **출시 시 기능 5개로 제한** — Mem 실패 회피
3. **iOS 우선** — Android는 출시 후 2~3개월 뒤
4. **한국어 OCR 정확도 핵심 모트** — Vision Framework 한국어 보강, 자체 후처리 필수
5. **첫 다운로드 마찰 최소화** — 첫 7일 무료 + 첫 화면 가치 즉시

#### 차별화 포인트 (한 줄)
> **"한국 사람이 인스타·카톡·유튜브 스크린샷 찍는 순간, 알아서 정리하고 캘린더까지 잡아주는 iOS 외장 두뇌"**

- vs mymind: 한국어 OCR + 구글 캘린더 + 가격 1/3
- vs Mem 2.0: 모바일·자동 감지 우선, 야심 작게 시작
- vs Sumind.app: 한국어 + 캘린더 OCR + iOS 친화
- vs Apple Notes: 큐레이션·요약·구글 캘린더 결합
- vs Readwise: 긴 글 아닌 짧은 시각적 캡처
- vs Notion/Obsidian: 정리 사용자 부담 0
