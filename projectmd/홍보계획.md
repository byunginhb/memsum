# 📣 홍보 계획 — Memsum

> 카톡 채널 제거. **사전 판매 4주 검증 → 12주 iOS 빌드 → 출시** 의 3단계 전략.
> 인스타 릴스가 메인 채널. "사진 앨범 스크린샷 1,800장" 후킹.
> 이름 의미를 활용: **Memsum = Memory + Summary**

---

## 1. 타깃 고객 프로파일 (페르소나 노트 요약)

- **Primary**: 이수현(34세, 외국계 IT 마케팅) — 사진 앨범 스크린샷 1,847장 보유, 노션 한 달 만에 포기
- **Secondary**: 박지훈(28세, 1인 뉴스레터) — 한국어 비즈니스 자료 OCR 페인
- **Anti-persona**: 정노션(30세, PM, PKM 헤비유저) — **의도적 배제**

## 2. 핵심 메시지

### 1) 한 줄 카피 (3종 A/B 테스트)

| 코드 | 카피 | 사용처 |
|------|------|--------|
| **H1** | "찍기만 하세요. Memsum이 알아서." | 메인 / 랜딩 / 앱 첫 화면 |
| **H2** | "사진 앨범 스크린샷 1,800장. 다시 본 적 있으세요?" | 인스타 릴스 후킹 (1초) |
| **H3** | "스크린샷에서 약속 감지 → 구글 캘린더 자동 등록" | 시그니처 기능 강조 |

### 2) 이름 의미 활용한 메시지

> **Memsum = Memory + Summary.**
> 당신의 기억(Memory)을 매주 5줄로 요약(Summary)해 다시 보내드립니다.

이 문장 자체가 좋은 마케팅 자산. 어원이 곧 가치 설명.

### 3) 메시지 구조 (페인 → 솔루션 → 시그니처 → 증거 → CTA)

```

[시그니처]

[증거]
"사전 결제 ○○명 / 첫 베타 만족도 ○○점

[행동 유도]
"첫 7일 무료 + 사전 결제 평생 50% 할인 (₩2,450/월) — 첫 100명 한정
👉 App Store에서 'Memsum' 검색"
```

### 4) 카테고리 정의 문구 (PKM 헤비유저와 선 긋기)
> ⚠️ **"Memsum은 노션·옵시디언 대안이 아닙니다."**
> 이미 PKM 시스템 잘 쓰시는 분께는 추천하지 않습니다.
> Memsum은 "스크린샷 폴더가 1,000장 넘어가는데 정리 못 하는 사람들"을 위한 도구입니다.

### 사전 판매 / 한정 프로모션
- **사전 결제 ₩2,450/월 (Lite 평생 50% 할인)** — 첫 100명 한정

### 전환 트리거 (자동화)
1. **Free 용량 80% 도달** → "이번 주에만 380MB 사용. Lite로 무제한" 푸시
2. **첫 5줄 리포트 만족도 4+ 평가** → 즉시 Lite 업셀 CTA
3. **누적 캡처 100개 돌파** → "당신의 외장 두뇌가 자산이 됐어요. 더 큰 용량 잠금 해제"
4. **첫 캘린더 자동 등록 성공** → "이게 진짜 마법이죠? 무제한으로 쓰시려면 Lite"
5. **30일 사용자 자동 리포트** → 한 달 캡처 그래프 + 가치 시각화 + Pro 추천

### 해지 방지
- **사용자별 "내 외장 두뇌 자산" 시각화** — 누적 캡처·연결·재발견·캘린더 등록 수

### 1차 No-Go 시그널 (피벗·중단 기준)
- **사전 판매 100명 미달 (4주)**: 빌드 중단, 페르소나·메시지·기능 전면 재검토 또는 아이디어 폐기
- **출시 첫 주 다운로드 < 1,000**: 인스타 릴스 후킹 메시지 즉시 재작성 + 인플루언서 유료 협업
- **30일 5줄 리포트 만족도 < 60%**: 분류·요약 모델 재학습 + GPT-4o 업그레이드
- **30일 캘린더 자동 등록 OCR 정확도 < 85%**: 한국어 OCR 후처리 모델 재학습 (이게 시그니처라 정확도 핵심)
- **30일 활성 사용자 비율 < 30%**: 권한 요청 흐름 재설계 + 첫 화면 가치 즉시 강화
- **Apple Intelligence 동일 기능 발표**: 한국어·구글 캘린더·Ask Your Brain 차별화 전면 강화 + 가격 인하 검토
