## Memsum — 찍기만 하면 알아서 정리되는 외장 두뇌

> **이름**: Memsum (멤섬) — **Mem**ory + **Sum**mary
> **로고**: 시안 D — 대칭 모노그램 (MEM/SUM)
> **자세히**: [[아이디어/Memsum/브랜드|브랜드 가이드]]

### 한 줄 정의
> 인스타·카톡·유튜브 스크린샷 찍는 순간, 알아서 정리하고 캘린더까지 잡아주는 iOS 외장 두뇌.

### 핵심 슬로건
> **"찍기만 하세요. Memsum이 알아서."**

### 핵심 기능 5개 (출시 시점 — 한 번에 풀 빌드)
1. **스크린샷 자동 감지** — 찍는 순간 푸시 "Memsum에 저장할까요?" 1탭
2. **날짜 OCR → 구글 캘린더 자동 등록 제안 (옵션)** — 스크린샷에 일정 있으면 한 번에 캘린더 추가 (시그니처 기능)
3. **공유 시트 익스텐션** — 인스타·사파리·트위터에서 1탭 캡처
4. **자동 주제 분류 + 검색** — 폴더·태그 없이 자동 그룹핑, 자연어 검색
5. **주간 5줄 리포트** — 일요일 저녁 9시 푸시, 이번 주 핵심 5개 (**"Sum"** 기능)

### 보조 캡처 경로
- Siri 단축어 ("헤이 시리, 멤섬")
- 홈화면 위젯 (1탭 음성·텍스트 메모)
- 사진 라이브러리 OCR (찍은 사진도 자동 인식)

### 출시 후 확장 (Month 2~3)
- Android 버전
- 사파리·크롬 확장 (데스크탑)
- Apple Watch
- 이메일 포워딩 (capture@memsum.com)

### 1. Software 3.0 적합도 (카파시 렌즈)

- **AI 역할**: OCR(이미지 텍스트 추출) + 분류(자동 주제 묶음) + 추출(날짜·이벤트·장소) + 요약(주간 5줄) + 검색(개인 RAG)
- **인간 검증 가능 여부**: 매우 쉬움 — 모든 AI 출력이 사용자 앞에 보여지고 즉시 [예]/[아니오]
- **Vibe Coding MVP 가능**: **부분적** — iOS 네이티브는 Vibe Coding으로 어렵고 Swift 전문 지식 필요. 단 백엔드(GPT API + Supabase)는 Vibe Coding 가능
- **부분 자율성 설계 ✅**: AI가 감지·추출·분류·요약, 사용자는 [예] 한 번. 검증 부담 최소
- **Jagged Intelligence 회피 ✅**: OCR·분류·요약 다 AI 잘하는 영역. 사용자가 즉시 검증
- **카파시 부합도**: 자연어 인터페이스 + 자동 시스템 변환. **"앱 안 켜도 일어나는 백그라운드 AI"** 가 핵심 진화

### 2. 시장 검증 (마이크 렌즈) — 시장레이더 결과 반영

- **경쟁 제품 존재**: 있음
  - iOS 자동 캡처: mymind ($13), Mem 2.0 ($14.99)
  - 동일 컨셉 90%: **Sumind.app** ("Stop saving links you'll never read") ← 강력한 시장 검증
  - Resurfacer: Readwise
  - 노트: Apple Notes, Notion, Obsidian
- **시장 검증 완료 ✅**: 수십만 유료 사용자, 수십억 달러 펀딩
- **차별화 포인트**:
  - **시그니처: 스크린샷 한국어 OCR → 구글 캘린더 자동 등록** (시장에 없음)
  - 한국어 정확도 (Vision Framework + 자체 후처리)
  - 가격 ₩4,900 (글로벌 1/3)
  - **용량 기반 구독제 = 자산 누적 락인**
  - **이름의 의미**: Memory + Summary 결합이 제품 가치 직접 전달
- **AI 의존도**: 핵심 (OCR·분류·추출·요약 모두 AI)
- **외부 API 없이 핵심 가치**: 불가 (GPT API + Google Calendar API 필수)
- **AppSumo LTD 가능**: Month 4~6 검토 (글로벌 확장 시점)
- **MRR 전환**: 매우 자연스러움 (₩4,900~₩19,900 4단 플랜)
- **대기업 위험**: Apple Intelligence 추격이 최대 리스크. 한국어 + 구글 캘린더로 방어

### 3. 실행 가능성

- **사전 판매 검증**: **4주 (의무)**
  - Week 1: Figma 프로토타입 + 데모 영상 제작
  - Week 2: 랜딩 페이지 + 사전 결제 ₩3,000 (평생 50% 할인) Toss Link
  - Week 3: 인스타 릴스 + 디스콰이엇 + 트위터 발사
  - Week 4: **사전 결제 100명 미달 시 빌드 중단** / 100명 이상이면 빌드 시작
- **MVP 빌드**: **12주 (iOS만)**
  - Week 1~2: iOS 프로젝트 셋업 + 스크린샷 감지 (PHPhotoLibraryChangeObserver) + 권한 흐름
  - Week 3~4: OCR 파이프라인 (Vision Framework + GPT 후처리) + 한국어 정확도 튜닝
  - Week 5~6: 공유 시트 익스텐션 + Siri 단축어 + 위젯
  - Week 7~8: 자동 분류 (임베딩 + 클러스터링) + 검색 UI
  - Week 9~10: 구글 캘린더 OAuth + 자동 등록 흐름
  - Week 11: 주간 5줄 리포트 (백엔드 cron + 푸시)
  - Week 12: 결제 (Apple In-App Purchase + Toss) + TestFlight 베타 → App Store 출시

- **첫 수익 경로 (사전 판매 + 출시 후)**:
  - **인스타그램 릴스** — 메인 채널, "스크린샷 1,800장" 후킹
  - **디스콰이엇** — 한국 메이커/얼리어답터, 메이커 로그
  - **유튜브 PKM·마케팅 채널 협업** — 이연·박정후 등
  - **트위터 한국 AI/마케팅** — 빌드인퍼블릭
  - **긱뉴스** — 한국 개발자/얼리어답터

- **목표 월 수익**:
  - Month 0 (사전 판매): 사전 결제 100명 × ₩3,000 = ₩30만 (검증 통과 기준)
  - Month 1 (출시): 200명 × ₩4,900 = ₩100만
  - Month 3: 700명 × ₩4,900 + 100명 × ₩9,900 = ₩440만
  - Month 6: 2,200명 × ₩4,900 + 400명 × ₩9,900 + 50명 × ₩19,900 = ₩1,575만
  - Month 12: 7,500명 + 다양한 플랜 분포 = ₩5,000만+

### 4. 리스크 체크

- **99% 실패 패턴 해당 여부**: 부분 해당 → 사전 판매 검증으로 회피
  - 위험 1: **PMF 검증 전 12주 빌드 = Mem 실패 길**
    - 해결: **사전 판매 4주 검증 의무화** — 100명 미달 시 빌드 중단
  - 위험 2: **Apple Intelligence 추격 = 시그니처 기능 잠식 위험**
    - 해결: 한국어 OCR 정확도 모트 + 구글 캘린더 통합 (Apple 안 함) + Pro 플랜(Ask Your Brain)으로 가치 확장
  - 위험 3: **iOS 한정 = 시장 절반 잃음**
    - 해결: Month 3~4에 Android 출시 (한국 iOS 시장만 먼저 검증)
  - 위험 4: **앱 다운로드 마찰 = 카톡 채널보다 큼**
    - 해결: 인스타 릴스 후킹 강화 + 첫 화면 가치 즉시 + 첫 7일 무료
  - 위험 5: **개발 비용 = 1인 12주 또는 외주 ₩2,000~5,000만**
    - 해결: 사전 판매 ₩30만 + 부트스트랩 / 외주 시 사전 검증 후 시작
  - 위험 6: **Sumind.app 직접 경쟁**
    - 해결: 한국어 + 캘린더 OCR + 멤섬 한국 친화 브랜드

- **가장 큰 리스크**: **Apple Intelligence가 한국어 OCR + 캘린더 자동 등록 발표**
- **모트 3가지**:
  1. **한국어 OCR 정확도** — Vision은 한국어 약함, 자체 후처리 (GPT-4o-mini) 모트
  2. **구글 캘린더 통합** — Apple은 자사 캘린더만 강화, 구글 안 함
  3. **자산 누적 락인** — 6개월+ 사용자는 100GB 자산 쌓여서 해지 어려움
